# 平台用工连续劳务个税计算器
## 产品目标
给零工使用的一个简单的个税计算器，可以方便的计算税额。
## 连续劳务累计扣除法
### 一个自然年内，连续获得劳务报酬的月份为一个连续周期
比如：1月获得报酬，3月和4月获得报酬，则1月为一个连续周期，3月和4月为新的连续周期。新自然年重置为新的一个连续周期。
### 按月收入进行计算，计算逻辑是：
当月应预扣预缴税额 =（本期累计收入-本期累计费用-本期累计减除费用）×预扣率-速算扣除数-本期累计已预扣预缴税额。
- 本期累计收入：本期所有月份的累计收入
- 本期累计费用：固定为本期累计收入的20%
- 本期累计减除费用：固定为本期每个月5000元
- 本期累计已预扣预缴税额：是指本期所有月份的已预扣税额之和
### 七级累进税率表
```javascript
[
      { max: 36000, rate: 0.03, deduction: 0 },
      { max: 144000, rate: 0.1, deduction: 2520 },
      { max: 300000, rate: 0.2, deduction: 16920 },
      { max: 420000, rate: 0.25, deduction: 31920 },
      { max: 660000, rate: 0.3, deduction: 52920 },
      { max: 960000, rate: 0.35, deduction: 85920 },
      { max: Infinity, rate: 0.45, deduction: 181920 }
    ]
```
例如：当月收入在36000元以内的，那么预扣率rate就是0.03，速算扣除数deduction就是0

## 页面功能模块
### 标题
平台用工连续劳务个税计算器
### 信息汇总区域
显示全年的汇总信息：
- 总收入
- 总税后收入
- 总税额
- 平均税率
### 主体内容区域
#### 显示一个自然年1月至12月，每月一行
#### 分为4列
- 第一列显示月份
- 第二列让用户填写当月收入
- 第三列显示计算后的当月应预扣预缴税额
- 第四列显示计算后的当月税后收入
### 操作区
- 填充固定收入：让用户输入一个每月固定的收入，填充到12个月里面。
- 清空所有月份：清空现有所有月份里的收入。

## 页面设计风格
- 具有游戏化和趣味性
- 页面风格简洁明快

## 计算触发时机
当任意月份收入发生变动时，自动计算并显示相关数据
